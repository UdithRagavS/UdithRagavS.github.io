Projects

Highlighted projects showcasing practical application of my security and software development skills.

Sysmon + Splunk Detection Lab

Built a home blue-team lab streaming Windows endpoint telemetry into Splunk. Configured Sysmon (SwiftOnSecurity config) on a Windows VM, forwarded events with the Splunk Universal Forwarder, and used the Splunk Add-on for Microsoft Sysmon to parse logs into a dedicated index. Validated the pipeline with SPL over process-creation, network, and file events — surfacing command lines and parent-child process activity for threat hunting. Documented the full build in a write-up.

Blockchain for Smart Home Data Privacy (MSc Thesis)

Compared Ethereum (public) and Hyperledger Fabric (permissioned) for storing smart-home IoT data, evaluating privacy, performance, and cost. Encrypted a simulated dataset with AES-256-CBC and hashed activity data with SHA-256, stored encrypted payloads on IPFS with the CID referenced on-chain. Built a Solidity contract deployed to Sepolia via Hardhat, Alchemy, and MetaMask, plus a Node.js Fabric chaincode across a two-org network, with scripted checks for encryption, access control, and data exposure.

ML-Based Botnet Detection (CTU-13)

Coursework project applying machine learning to botnet detection on the CTU-13 dataset. Processed and labelled network-flow traffic, engineered features, and trained classification models to separate botnet activity from normal traffic, evaluating the results with standard detection metrics.

Role-Based Access Control (RBAC) Web Platform

Built an RBAC web platform using JavaScript, Node.js, and PostgreSQL in a microservices structure. Implemented role-based permissions, input validation, and automated tests (Jest, Cypress) to validate access-control behaviour and core flows.

Crypto Dashboard

Built an interactive crypto dashboard using JavaScript, React, and TypeScript, integrating the CoinGecko API. Delivered dynamic Chart.js visualizations and filtering to track real-time prices, with caching and responsive design for a smooth user experience.
